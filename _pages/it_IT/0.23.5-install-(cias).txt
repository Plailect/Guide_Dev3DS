### Istruzioni

1. Estrai i file all'interno dell'archivio `.zip` del 0.23.5 SystemUpdater nella root della tua scheda SD
1. Reinserisci la scheda SD nella tua console PANDA
1. Attraverso il Dev Menu, apri la cartella del 0.23.5 SystemUpdater
1. Premi insieme (L + R + A) per installare tutti i file CIA
1. Trova il file NATIVE_FIRM CIA
  + CTR / SPR / FTR: `0004013800000002.cia`
  + SNAKE / CLOSER / JAN: `0004013820000002.cia`
1. Aggiorna la FIRM della console PANDA tenendo premuto (Start + Y) mentre è selezionato il file NATIVE_FIRM CIA
