### Lectură obligatorie

Cu acest ghid veți putea: 

- Instala versiunea 0.23.5 pentru regiunea / dispozitivul dumneavoastră
- Instala boot9strap folosind CakeHax Browser
- (Opţional) instala cea mai recentă versiune pentru regiunea / dispozitivul dumneavoastră

### Lectură obligatorie

Există două metode diferite de a instala versiunea 0.23.5 în unitatea PANDA.

O metodă nu necesită instrumente externe, dar acceptă numai unele regiuni şi dispozitive din cauza pachetelor CIA lipsă pentru restul regiunilor şi dispozitive.

O altă metodă acceptă toate regiunile şi dispozitivele, dar necesită un flash card CTR şi hardware-ul necesar de dezvoltare pentru a scrie un CSU în acel flash card CTR.

___
### Metode

___

#### Instalare 0.23.5 (CIA-uri)

Această metodă este compatibilă cu următoarele dispozitive 

+  SNAKE / CLOSER / JAN
    + [EUR / JPN / USA]
+  CTR / SPR / FTR
   + [EUR / USA]

### Continuați la [Instalare 0.23.5 (CIA-uri)](0.23.5-install-(cias))

___

#### Instalare 0.23.5 (CSU-uri)

Această metodă este compatibilă cu următoarele dispozitive 

+  SNAKE / CLOSER / JAN
   + [EUR / JPN / USA]
+  CTR / SPR / FTR
   + [CHN / EUR / JPN / KOR / TWN / USA]
